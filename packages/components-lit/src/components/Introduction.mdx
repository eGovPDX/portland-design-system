# Portland Component Library - Lit Components

Welcome to the **Portland Component Library** built with [Lit](https://lit.dev/)! This collection provides fast, lightweight web components that work across all modern frameworks.

Lit is a simple library for building fast, lightweight web components. It provides:

- **Small & Fast**: ~5KB runtime with excellent performance
- **Standards-based**: Built on Web Components standards
- **Framework Agnostic**: Works with React, Vue, Angular, or vanilla JavaScript
- **Developer Experience**: Great TypeScript support and dev tools

- **Universal Compatibility**: Use the same components across different frameworks

- **Server-Side Rendering**: Supports SSR and static site generation
