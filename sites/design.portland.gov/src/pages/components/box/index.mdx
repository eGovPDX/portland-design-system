# Box

The Box component is a foundational layout surface for grouping related content.
It applies consistent spacing, color, and contrast styles across the design
system.

Use Box when you need to:

- Group content into a clear visual section
- Apply a semantic color context (default, primary, success, warning, etc.)
- Build nested surfaces while keeping consistent component styling
- Create a new component that needs accessible colors

## Properties

The combination of `color` and `variant` controls the background, content, and
border color for each use. This allows you to choose from accessible color
patterns quickly.

### As

Box is rendered as a `div` by default but can be changed to any semantic element with the `as` prop. For example, use `section` for a standalone content area or `aside` for tangential information.

## Usage notes

- Choose the variant based on emphasis needed: start with `subtle` or `moderate`
  for most layouts.
- Reserve `emphasis` and `strong` for callouts or high-attention regions.
- Keep content hierarchy clear by combining Box with semantic elements such as
  `section`, `article`, and `aside`.
